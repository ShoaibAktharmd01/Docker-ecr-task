# Dockerize Simple Application and Push to AWS ECR

## Project Overview

This project demonstrates how to containerize a simple Python Flask application using Docker and push the container image to AWS Elastic Container Registry (ECR).

## Technologies Used

- Python
- Flask
- Docker
- AWS ECR
- GitHub

## Files

- app.py
- requirements.txt
- Dockerfile
- README.md

## Build Docker Image

```bash
docker build -t hello-ecr-app .
